# 💕 Kanaya Shafa Aulia — Photo Album

A beautiful digital photo album / scrapbook website.

## 📸 Cara Tambah Foto

1. Simpan foto ke folder `photos/`
2. Buka file `app.js`
3. Cari bagian `PRELOADED_PHOTOS` dan tambahkan foto:

```js
const PRELOADED_PHOTOS = [
    { file: 'foto1.jpg', caption: 'Kenangan pertama kita 💕' },
    { file: 'foto2.jpg', caption: 'Momen bahagia bersama ✨' },
    { file: 'foto3.jpg', caption: 'Date night 🌙' },
];
```

4. Commit & push, selesai!

## 🚀 Deploy ke GitHub Pages

1. Go to **Settings** > **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **(root)**
4. Save, tunggu beberapa menit
5. Website live di `https://286rx.github.io/Kanaya/`

## ✨ Features

- 🏠 Hero section dengan animasi gradient pink-rose gold
- 📸 Upload foto (drag & drop)
- 🖼️ Galeri foto (grid & masonry view)
- ✏️ Edit caption per foto
- 🔍 Lightbox full-size view
- 💕 Floating hearts animation
- 📱 Responsive design
